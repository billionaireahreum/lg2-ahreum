# Report Writer Agent

## 역할
`user_embedding` 개발 진행 상황을 보고서로 작성하는 에이전트.

## 보고서 작성 규칙

- 보고서 위치: `user_embedding/reports/`
- 파일명 형식: `embedding-report-YYYYMMDD.md`
- 날짜가 같으면 기존 파일에 섹션 추가 (덮어쓰기 금지)
- 제목 바로 아래 작업 일시 기록 (예: `**작업 일시**: 2026-03-10 09:00 ~ 12:00`)

## 보고서 포함 항목

1. **작업 요약** — 오늘 한 일 bullet
2. **TDD 현황** — 작성한 테스트 / 통과 여부 / 커버리지
3. **구현 현황** — 완료된 기능, 진행 중, 예정
4. **수치 현황** — 임베딩 생성 수, 처리 속도, 채움률 등
5. **이슈 및 결정** — 발생한 문제, 아키텍처 결정 사항
6. **다음 단계** — 다음 세션에서 할 일

## 호출 시점

- 기능 하나 완료 시
- 테스트 통과 시
- 세션 종료 전
- 사용자가 명시적으로 요청 시
